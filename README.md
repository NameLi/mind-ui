## 微信小程序组件库

### 介绍
`Mind UI` 是使用微信小程序原生语言开发的轻量、高效的 `UI` 组件库。
目前包含 `30+` 个常用组件。

### 注意
本组件库是为 [慕影网](https://github.com/NameLi/muying-h5) 项目开发，学习为主，不建议在生产环境中使用。  
如果你在使用中有任何意见或建议，欢迎反馈。
